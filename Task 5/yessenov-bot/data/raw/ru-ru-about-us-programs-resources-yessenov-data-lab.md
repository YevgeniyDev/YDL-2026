Летняя школа для действующих аналитиков данных и планирующих работать в этой сфере. Программа открылась в 2018 году и представляет из себя длящийся несколько недель интенсив (в последние годы – онлайн) по обучению казахстанских молодых ученых/специалистов базовым навыкам анализа больших данных для применения ими этих знаний в дальнейшей работе.

- Yessenov Data Lab 2026 20 грантов для молодых ученых в летнюю школу для аналитиков. Yessenov Data Lab 2026
